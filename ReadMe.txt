Реализованный функционал SPA:

1. При загрузке SPA появляется рисунок со слоупоками, идет загрузка
2. По завершении запроса, отображаются карточки покемонов без сортировки, информация
   о конкретном покемоне доступна при переходе по кнопке на карточке.
3. После загрузки становятся доступными кнопки навигации, которые позволяют отсортировать
   покемонов по типу: покемон, тренер, энергия. Также доступна кнопка "все карты", тогда
   отображаются опять все карточки, но без запроса на сервер.
4. При нажатии на кнопку "перезагрузка карт" идет запрос на API сервера, грузятся снова ВСЕ карты.
5. При переходе на конкретного покемона идет роутинг на его страничку. Отображается большое фото,
   некоторая информация из профиля (можно отобразить что угодно из файла, вопрос потребности).
   Кнопки в навигации скрываются при переходе, доступно 2 опции - вернуться обратно, перезагрузить
   покемона.
6. При перезагрузке этого покемона реализован второй набор функций, который делает короткий
   запрос непосредственно на ЭТОГО покемона, он грузится значительно быстрее, и его сохраняем
   в отдельный стейт.
